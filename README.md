# 🏭 Gestão de Peças e Qualidade Industrial

Este projeto é um protótipo funcional para automação de inspeção em linhas de montagem, desenvolvido para o Desafio de Automação Digital.

## 🚀 Funcionalidades
- **Cadastro Automatizado:** Geração de ID sequencial e persistência em SQLite.
- **Validação de Qualidade:** Motor de regras para peso (95g-105g), cor (Azul/Verde) e comprimento (10cm-20cm).
- **Logística de Armazenamento:** Agrupamento automático de peças em caixas com capacidade de 10 unidades.
- **Relatório Gerencial:** Visão consolidada de aprovadas, reprovadas e motivos de falha.

## 🛠️ Como Rodar o Projeto
1. Tenha o Python 3.11+ instalado.
2. No terminal, execute: `python main.py`
